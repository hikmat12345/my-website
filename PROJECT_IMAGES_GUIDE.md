## Image Specifications

**Recommended Image Format:**
- **Format:** PNG or JPG
- **Dimensions:** 500x300px minimum (aspect ratio 16:9 or similar)
- **Size:** Under 200KB per image for optimal performance
- **Quality:** High-quality screenshots or mockups of the projects

## How to Add Images

1. Prepare your image files (screenshot, mockup, or logo for each project)
2. Name them exactly as specified above
3. Place them in the `public/images/` directory
4. No code changes needed - the links are already configured!

## How Images Are Used

- **Portfolio Section:** Full-size display with project details
- **Responsive:** Automatically scales for mobile, tablet, and desktop
- **Hover Effects:** Images zoom slightly on hover for better interaction
- **Performance:** Next.js Image component optimizes for web

## Testing Your Images

After adding images, check:
1. All images load without 404 errors
2. Images display correctly in the Portfolio section
3. Images are responsive on mobile devices
4. Click on project cards to see details and links
